# Testorium


The project is a reference sample of a new Flutter app

Features:

 <ul>
  <li>internationalization</li>
  <li>routing (auto_route)</li>
  <li>bloc, get_it</li>
  <li>themes changing on the fly</li>
  <li>fonts</li>
  <li>shimmer effect while loading</li>
</ul>
